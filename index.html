Combined Hierarchy

Fully customizable hierarchical visualization component

A reusable component for Salesforce© Lightning Experience to show hierarchies of objects and related records in the same view.

The Combined Hierarchy component allows you to display hierarchical relationships between objects (both standard and custom) in a collapsible tree with related records in tables that can be sorted by any field.

The Combined Hierarchy component can be adapted for any standard or custom object that has a hierarchical relationship to itself (e.g. Accounts, Campaigns, Cases, etc.).

It is easy to use, just drag and drop the component on the Lightning page you want (Record, App, Community) and configure the fields to be shown in the visualization. That's all you need to do.

Notifications

Real-time notifications component

A reusable component for Salesforce© Lightning Experience to show custom notifications in your custom Apps.

The Notifications component allows an administrator/developer to include advanced notifications on their custom pages of any Salesforce application.

The Notifications component can be adapted for multiple use cases, from real-time notification of tasks running in the background to user-to-user communication.

It is easy to use, just drag and drop the component on the Lightning page you want (Record, App) and configure the style and the context. That's all you need to do.

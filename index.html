Combined Hierarchy

A reusable component for Salesforce Lightning Experience to show a hierarchy of objects and their related records in the same view.

The Combined Hierarchy component allows the hierarchical relationships of your objects (standard and custom) to be displayed in a collapsible tree that includes related records in a table that can be sorted by any field.

Use Cases

Here you can find some example configurations and use cases that can be solved using Combined Hierarchy component by ARES.

  • Accounts with Opportunities
  • Contacts with Cases
  • Cases with Attachments
  • Users with Approvers
  • Custom Object with custom relationships!

Endless possibilities. The Combined Hierarchy component can be adapted for any standard or custom object that has a hierarchical relationship to itself.

It is easy to use, just drag and drop the component on the Lightning page you want (Record, App, Community) and configure the fields to be shown in the hierarchy tree and in the table. That's all you need to do.
